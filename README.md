# Kawr Business Beta 1.0

## Sobre o projeto

O Kawr Business é uma plataforma de apoio à gestão financeira e tributária desenvolvida para auxiliar empresários e contadores na análise de informações financeiras de forma simples, organizada e transparente.

O sistema não substitui um contador. Seu objetivo é facilitar a visualização dos dados e apoiar a tomada de decisões.

---

## Funcionalidades

- Dashboard para Empresário
- Dashboard para Contador
- Score Financeiro (0 a 100)
- Alertas Financeiros
- Cálculo estimado do DAS
- Cálculo do Fator R (MVP)
- Identificação de Anexo (MVP)
- Memória de Cálculo
- Interface responsiva
- Gráficos financeiros (em desenvolvimento)

## Aviso

O Kawr Business é uma ferramenta de apoio à gestão.

Os cálculos apresentados são estimativas para fins de demonstração nesta versão Beta e não substituem a análise de um contador habilitado nem garantem conformidade fiscal.
